# Qiuzhi Data Analyst

## Description
A specialized skill for analyzing and summarizing CSV data. Uses pandas and AI to transform raw data into actionable insights.

## Usage
Use this skill when the user wants to:
- Analyze CSV files
- Generate data summaries
- Create visualizations from data
- Identify patterns in datasets

## Tools
- `read`: Read CSV files
- `exec`: Run pandas/python analysis
- `web_search`: Find related information
- `write`: Save analysis results

## Scripts
- `scripts/csv_analyzer.py`: Main CSV analysis script
- `scripts/data_visualizer.py`: Data visualization script
