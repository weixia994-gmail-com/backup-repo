# Qiuzhi Data Analyst

A specialized module for analyzing and summarizing CSV data within the Qiuzhi Project.

## Features
- CSV file analysis and summarization
- Data pattern identification
- Missing value detection
- Numerical data statistics

## Structure
- `SKILL.md`: Skill definition for OpenClaw
- `scripts/csv_analyzer.py`: Main analysis script
- `references/`: Related documentation
- `assets/`: Static assets and templates

## Usage
```bash
python3 scripts/csv_analyzer.py path/to/your/file.csv
```

This will generate an analysis report in Markdown format.
