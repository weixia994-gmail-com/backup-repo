# Qiuzhi Video Maker - Requirements

## Core Dependencies
- text-to-speech (TTS) for narration
- video editing capabilities
- background music integration
- visual elements and rendering

## Installation Script
```bash
# Install required packages
pip install gtts pygame pydub moviepy
```

## Feature List
- Convert text scripts to audio narration
- Combine audio with background music
- Add visual elements to create video
- Render final video files in MP4 format
