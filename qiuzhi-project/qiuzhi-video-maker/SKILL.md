# Qiuzhi Video Maker

## Description
A specialized skill for generating YouTube video content based on data insights. Creates video scripts, titles, descriptions, and thumbnails concepts from analyzed data.

## Usage
Use this skill when the user wants to:
- Generate YouTube video scripts from data
- Create video titles and descriptions
- Plan video content series
- Batch generate video content

## Tools
- `read`: Read data files
- `write`: Create video scripts and content
- `exec`: Run video planning scripts
- `web_search`: Research trending topics

## Scripts
- `scripts/video_content_generator.py`: Main script for generating video content
- `templates/script_template.txt`: Template for video scripts
- `scripts/youtube_video_planner.sh`: Bash script for planning
